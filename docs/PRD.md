# PRD

## 项目目标

做一个“女神人格 Skill 生成器”。

用户输入一句话人设、聊天片段、截图或场景描述，系统生成一个可直接调用的独立 Skill，用来稳定扮演某种女神人格。

## 核心价值

1. 把模糊标签变成可执行的人格规则
2. 支持多种常见“女神感”原型混合
3. 支持后续追加纠偏和迭代

## 典型输入

- “做一个高冷女神，爱答不理，偶尔会回一句”
- “像三无学姐，冷淡但观察很细”
- “偏女王，不主动，不喜欢解释”
- 一段聊天记录或截图

## 典型输出

生成 `goddesses/{slug}/`：

- `memory.md`
- `persona.md`
- `meta.json`
- `SKILL.md`

## 系统结构

### Part A: Context Memory

保存稳定背景信息：

- 场景
- 身份关系
- 话题素材
- 红线
- 触发温度变化的条件

### Part B: Persona

保存真正驱动回复的规则：

- 原型混合
- 说话方式
- 回复长度和节奏
- 情绪升降
- 边界感

## 支持的主原型

1. 冰山高冷
2. 疏离回避
3. 傲娇嘴硬
4. 三无淡感
5. 女王支配
6. 若即若离

## 成功标准

1. 读起来像具体的人
2. “高冷”“不理睬”能落实为明确互动行为
3. 支持后续纠偏
4. 最终 Skill 可以直接运行
