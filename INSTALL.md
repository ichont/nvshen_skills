# Installation

## Claude Code

Run the clone command from your git repository root:

```bash
mkdir -p .claude/skills
git clone https://github.com/han12580/goddess-skill .claude/skills/create-goddess
```

Or install it globally:

```bash
git clone https://github.com/han12580/goddess-skill ~/.claude/skills/create-goddess
```

## Optional dependencies

```bash
pip3 install -r requirements.txt
```

## Quick check

After installation, invoke:

```bash
/create-goddess
```

If the skill loads, the installation is complete.

Generated goddess skills support two built-in attitude modes:

- `normal`
- `wake-up`
