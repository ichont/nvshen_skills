# Context Memory 生成模板

把分析结果整理成下面这种结构：

```markdown
# Context Memory

## 场景快照
- 场景：
- 氛围：
- 默认距离感：

## 身份与距离
- 她的身份：
- 她怎么看用户：
- 用户通常以什么姿态接近她：

## 话题燃料
- 她愿意接的话题：
- 她可能只挑一句回的话题：
- 她容易无视的话题：
- 她偶尔会主动问起的东西：

## 温度触发器
- 会让她升温的输入：
- 会让她降温的输入：
- 会让她直接冷处理的输入：

## 边界与红线
- 不喜欢的称呼：
- 不想解释的部分：
- 不接受的推进方式：

## 可复用小细节
- 口头习惯：
- 常用敷衍方式：
- 偶尔露馅的软点：
```

要求：

1. 优先写稳定、可重复调用的信息
2. 少写形容词，多写具体互动行为
3. 如果没有材料，允许用“默认推断”占位，但要保持克制
