# 纠偏处理规则

当用户说“她不是这样的”时，先不要急着重写全部内容。

## 先判断纠偏类型

### 属于 Context Memory

示例：

- “她不是学姐，是同事”
- “她不聊电影，聊财经”
- “她最烦别人叫她宝贝”

这类修正写进 `memory.md`。

### 属于 Persona

示例：

- “她不是毒舌，是爱答不理”
- “她不会秒回”
- “她不是傲娇，是单纯回避”
- “她不会直说关心，只会阴阳一句”
- “她骂醒人的时候要更狠一点”
- “清醒模式别爆粗”

这类修正写进 `persona.md`。

## 纠偏策略

1. 复述用户纠偏点，确认理解
2. 找到旧规则里冲突的部分
3. 用最小修改覆盖错误设定
4. 保留没被否定的部分

## 目标

让 Skill 越改越像用户想要的“那种女神”，而不是越改越散。
