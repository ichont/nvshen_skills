# 信息录入引导

## 开场白

```text
你好，我来帮你做一个女神人格 Skill。

不用准备很复杂的材料。
给我一个代号、一点场景感，再加几个人格标签，就能先生成第一版。
后面还可以继续改冷一点、拽一点、嘴硬一点。
```

## 问题顺序

### Q1：代号

```text
先给她起个代号吧？

不一定是真名，可以是备注、外号、关键词。
比如：
- 冷妍
- 学姐
- 那位大人
- 白月光
- 77
```

### Q2：场景 / 身份感

```text
她大概是什么设定？

一句话就够，比如：
- 校园里的高冷学姐
- 办公室里不爱理人的前辈
- 线上认识的冷淡女生
- 一个虚构世界里的女王型角色
```

### Q3：人格原型和说话风格

```text
最后一句：你想要她是什么感觉？

可以直接扔标签，比如：
- 高冷、不理睬、慢热
- 三无、少话、很会冷场
- 女王、标准高、带审视感
- 傲娇、嘴硬、偶尔偷偷关心
- 若即若离、会吊着人、偶尔给甜头
```

### Q4：清醒模式强度（可跳过）

```text
她要不要带一个“骂醒 / 清醒模式”？

默认我会给她做两个模式：
- normal：平时
- wake-up：更锋利地把人骂清醒

你可以补一句你想要的力度，比如：
- 轻刺痛，但不爆粗
- 很冷，很直接，专门打断执念
- 女王式训斥，但不要低级脏话
```

## 可选补充

```text
- 聊天片段
- 截图
- 社媒文案
- 口头描述
- 世界观设定

没有也没关系，我可以直接开始。
```
